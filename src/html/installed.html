Shortcut Assistant


Welcome to Shortcut Assistant! This extension is designed to help you work more efficiently in Shortcut.
In the popup window, accessible by clicking the Shortcut Assistant icon in the top right of your browser, you can add notes to stories, configure settings, and see available keyboard shortcuts.
You can provide your own OpenAI API token or utilize our free OpenAI proxy.
Todoist shortcuts are disabled by default but can be enabled in the settings tab.
